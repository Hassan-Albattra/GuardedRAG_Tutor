# GuardedRAG Tutor

GuardedRAG Tutor is a safe Retrieval-Augmented Generation (RAG) system built with Python and Gradio.

## Features
- Secure RAG pipeline
- Document retrieval
- LLM-powered answers
- Guardrails for safe responses
